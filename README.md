
<h1>
Registro de Clientes usando o Xamarin
</h1>

<h2>
Getting Started
</h2>
<p>
Este projeto teste é voltado ao desenvolvimento e aprendizado baseado em desafios utilizando o Xamarin Forms durante o trabalho no LACTEC.
</p>
<h2>
App functions
</h2>
<pre>
<u1>
<li>Cadastro de clientes no banco de dados relacional</li>
<li>Importação e exportação de Json de Cadastro de usuários</li>
<li>Integração com servidor criado via CouchDB</li>
</u1>
</pre>
<h2>
Build With
</h2>
<pre>
<u1>
<li>Microsoft Visual Studio</li>
</u1>
</pre>
